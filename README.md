# andor-extended
Andor-Legenden Designer (Vue.js)

https://andor-extended.firebaseapp.com/


### Build local
Create all Files for the app in hosting/dist
```console 
cd hosting 
npm i 
npm run build
```

### Dev local
Start local Server with Hot-Module-Reloading
```console 
cd hosting 
npm i 
npm run dev
